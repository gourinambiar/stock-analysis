# Stock Price Analysis & Prediction Dashboard

An interactive dashboard for analyzing and predicting stock prices using technical indicators and machine learning.

## Features
- Candlestick charts with Moving Averages (MA20, MA50)
- RSI and Bollinger Bands indicators
- Next-day price prediction using Linear Regression
- R² of 0.92, MAE of $2.15 on AAPL test data

## Tech Stack
Python, Pandas, Scikit-learn, Plotly, Streamlit, yfinance

## How to Run
pip install -r requirements.txt
streamlit run app.py
